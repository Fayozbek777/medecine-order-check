# 🏥 Medicine Landing — Центр выкупа лекарств

Одностраничный сайт (лендинг) для компании по законному выкупу дорогостоящих лекарств у населения.

---

## 📋 О проекте

Сайт помогает людям продать неиспользованные препараты, оставшиеся после лечения, особенно в онкологии. Основная цель — быстро, безопасно и прозрачно вернуть часть потраченных средств.
